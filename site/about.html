SRE & DevOps Engineer

Quem Sou

Sou Marcos Sabatino, SRE e DevOps Engineer com foco em observabilidade, infraestrutura como código e engenharia de confiabilidade.

Acredito que bons sistemas precisam de bom código E boa operação. Observabilidade não é luxo — é necessidade.

Este Projeto

Observability Portal nasceu de uma convicção:

"A melhor forma de aprender observabilidade é ensinando enquanto você demonstra."

Então criei um site que não apenas fala sobre logs, métricas e traces, mas demonstra na prática usando Kubernetes, Prometheus, Grafana, Loki, Terraform, CI/CD — toda a stack moderna de observabilidade.

O site é:

  • 📚 Educacional — Ensina conceitos de observabilidade
  • 🏗️ Arquitetural — Demonstra stack profissional
  • 📊 Observável — Monitora a si mesmo em tempo real
  • 💻 Open Source — Todo código disponível no GitHub
  • 🎓 Portfolio — Mostra habilidades DevOps/SRE
